# Sawblade
A 3D-printable wheel. For use with Bruiser.

## Specs
- diameter: **54 mm**
- width: **32 mm**
- contact patch: **16 mm**

## Printing
- material: **95A TPU**
- line width: **0.4 mm**
- infill: **100%**
- shells: **3**
- flow: **105-110%** (adjust to fill gaps between lines)
- supports: **yes**
